# Harsha_python
This project analyzes the Red Wine Quality dataset using Python. It includes data loading, cleaning, and statistical analysis like mean, median, and variance. New features were created, and filtering techniques were applied to study how factors like alcohol and acidity affect wine quality.


This project highlights how chemical properties influence wine quality through statistical analysis.

- Alcohol content shows a positive impact on wine quality; higher alcohol wines tend to have better ratings  
- Acidity (fixed and volatile) affects taste balance; moderate levels contribute to better quality  
- pH levels play a role in stability and overall taste of wine  
- Most wines fall in the medium quality range (5–6), indicating average performance  
- High-quality wines (7 and above) are relatively rare in the dataset  
- Extremely low or high chemical values can negatively affect quality  
- Feature engineering helped uncover deeper relationships between attributes  
- Grouping and filtering techniques provided insights into patterns across different quality levels  

Overall, the analysis shows that wine quality is influenced by a combination of multiple chemical factors rather than a single parameter.
